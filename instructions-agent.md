# Instructions pour l'Agent - Intégration E-commerce (Bootstrap 5)

Tu dois intégrer les pages du site e-commerce de manière séquentielle, à raison d'une page par jour. Utilise strictement Bootstrap 5 (CDN ou fichiers locaux si présents) et un HTML5 propre et sémantique.

## Planning des tâches :
- **Jour 1 :** Interface de la Page d'accueil (Home page). Inclus un Header ( Barre de navigation responsive avec panier), un Hero Section (bannière dynamique), et une grille affichant les produits phares avec des badges Bootstrap (ex: "Promo", "Nouveau").
- **Jour 2 :** Page de Liste des Produits (Catalogue). Intègre des filtres sur le côté gauche (catégories, prix) et une liste de produits sous forme de cartes Bootstrap (`.card`) avec boutons "Ajouter au panier".
- **Jour 3 :** Page de Détail d'un Produit. Structure la page avec une zone d'images à gauche, le prix, la description détaillée, un sélecteur de quantité et les avis clients à droite.
- **Jour 4 :** Page Panier et Commande (Checkout). Crée un tableau récapitulatif des articles, une zone pour le code promo et un formulaire Bootstrap validé pour les informations de livraison.
- **Jour 5 :** Finalisation, responsive complet (mobile/tablette) et rédaction du guide de test dans `GUIDE_TEST.md`.

## Consigne de fin de tâche :
Chaque jour, après avoir terminé la page, écris un résumé détaillé de tes modifications dans `RECAP.md` (avec les classes Bootstrap importantes utilisées) et effectue un push sur la branche main.
